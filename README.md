# CraftkalOS

CachyOS base ISO with craftkal branding. Built via GitHub Actions.

## Usage

1. Go to **Actions** → **Build CraftkalOS ISO** → **Run workflow**
2. Wait ~25 min for build to complete
3. Download ISO from artifact

## Customization

Clone [CachyOS-Live-ISO](https://github.com/CachyOS/CachyOS-Live-ISO) and modify from inside the live system.
